# Bat-Monitoring-Site-Analysis

## Overview
This project analyses bat monitoring sites in Australia with data collected in 2025. While minimal in analysis, the project focuses more on data integration with two different but related datasets, and cleaning data from untidy to tidy.

## Repository Structure
```
.
├── Bat Monitoring Site Analysis.Rmd  # Code
├── Bat Monitoring Site Analysis.pdf  # Report
└── README.md
```

## Data Availability
The datasets used in the project is publicly available from the [Victorian Government](https://www.vic.gov.au/) website. It is not included in the repository but you may find the links in the references.

The dataset used in this project is publicly available and can be obtained from the original source

## Requirements
- R 4.5.x

## Getting Started
1. Clone the repository.
2. Download the datasets, links in references.
3. Run R markdown code
